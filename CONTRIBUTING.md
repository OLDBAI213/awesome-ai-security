# Contributing to Awesome AI Security

Thanks for your interest in contributing!

## How to Contribute

### Add a New Resource

1. Fork this repository
2. Add your resource to the appropriate section
3. Follow the format: `- [Name](URL) — One-line description`
4. Maintain alphabetical order within sections
5. Submit a Pull Request

### Quality Guidelines

- ✅ Resource must be relevant to AI security
- ✅ Link must be accessible
- ✅ Description must be concise and accurate
- ✅ Place in the correct category
- ❌ No paid-only tools without free tier
- ❌ No abandoned projects (no commits in 6+ months)

### Categories

| Section | What belongs here |
|---------|-------------------|
| AI-Powered Security Tools | Tools using AI/ML for threat detection, analysis |
| LLM Security | Securing Large Language Models |
| AI Agent Security | Security for autonomous agents |
| Vulnerability Research with AI | Using AI to find vulnerabilities |
| Prompt Injection Defense | Defending against prompt attacks |
| AI Red Teaming | Adversarial testing of AI systems |
| Papers & Research | Academic publications |
| Chinese Resources | Chinese-language tools and communities |

### Reporting Issues

- Broken links → Open an issue
- New section ideas → Open an issue
- Suggestions → Open an issue

## Code of Conduct

Be respectful and constructive. We're all here to make AI safer.
